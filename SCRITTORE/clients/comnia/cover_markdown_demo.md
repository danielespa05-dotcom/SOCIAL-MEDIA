# Ciao. Questo è un file .md

Sì, stai guardando un file Markdown.
Quella roba di cui parlano tutti ultimamente.

---

Puoi scrivere così → `**grassetto**` e diventa → **grassetto**

Puoi scrivere così → `*corsivo*` e diventa → *corsivo*

Puoi scrivere così → `# Titolo` e diventa un titolo enorme

---

Puoi fare liste:

- così
- e così
- e anche così

Oppure numerate:

1. prima cosa
2. seconda cosa
3. terza cosa

---

Puoi evidenziare codice → `come questo`

Puoi fare separatori → con tre trattini `---`

Puoi scrivere una citazione:

> "Le macchine non leggono tutto questo. Lo leggi tu."

---

**Premi ⌘ + Shift + V**

Vedrai la versione renderizzata.
Stessa identica cosa. Solo più bella per i tuoi occhi.
